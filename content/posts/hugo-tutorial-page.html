+++ date = '2025-05-01T15:36:10-07:00' draft = false title = 'How to use Hugo' comments = true tags = ["hugo", "static site", "blog"] categories = ["tutorials"] +++ This is how you use Hugo to write a new post! ## Step 1: Install Hugo You can install Hugo by following the instructions on the [Hugo installation page](https://gohugo.io/getting-started/installing/). ## Step 2: Create a new site You can create a new Hugo site by running the following command in your terminal: ```bash hugo new site my-hugo-site ```
